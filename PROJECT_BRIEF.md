# Acompaño - Mood Tracking for Neurodivergent People

## What is this?
Daily mood tracking app with Pattern Detective feature that translates raw data into narrative insights.

## Target User
Neurodivergent women 26-40 in LATAM. Experience high energy fluctuation, need self-knowledge without guilt.

## MVP Goal
Prove that <30 second check-ins can lead to meaningful pattern discovery.

## Phase 1 Features (This Sprint)
1. ✅ Daily check-in screen: Energy (1-5) + Emotion (3 emojis)
2. ✅ Data persistence (AsyncStorage)
3. ✅ History view of past check-ins
4. ⏳ Basic pattern: Weekly average energy

## Tech Stack
- **Framework:** React Native + Expo (SDK 52)
- **Language:** TypeScript (strict mode)
- **UI Library:** React Native Paper
- **State:** Zustand
- **Storage:** AsyncStorage (local only for MVP)
- **Navigation:** Expo Router
- **Testing:** Expo Go on phone

## Design Principles
- Dark mode default (reduce eye strain)
- Minimal animations (sensory sensitivity)
- Large touch targets (accessibility)
- No guilt-based language or metrics
- <30 seconds to complete check-in
